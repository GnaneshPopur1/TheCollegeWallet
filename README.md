# TheWallet

A centralized, secure, and intuitive platform to manage your entire financial ecosystem—from daily expenses and subscriptions to long-term student loans and tuition.

## Features
- **Financial Dashboard**: Net worth, cash flow, and credit score tracking.
- **Education Hub**: Tuition deadlines, financial aid, and student loan breakdowns.
- **Fixed Costs**: Rent/utility payment reminders and roommate split ledger.
- **Subscription Manager**: Auto-detect recurring charges and set cancellation reminders.

## Tech Stack
- Frontend: Angular 17+ (SCSS, Glassmorphism UI)
- Backend: Node.js, Express, PostgreSQL
- ORM: Prisma
- Security: JWT Authentication, AES-256

## Getting Started

### Backend
```bash
cd backend
npm install
npx prisma generate
node src/app.js
```

### Frontend
```bash
cd frontend
npm install
npm start
```
